# Contributing to 批量图片下载器

感谢你考虑为本项目做出贡献！

## 🤝 如何贡献

### 报告Bug
1. 在GitHub Issues中搜索，确保bug未被报告
2. 创建新Issue，使用清晰的标题和描述
3. 提供复现步骤、预期行为和实际行为
4. 如果可能，提供截图或错误日志

### 提出新功能
1. 在GitHub Issues中描述你的想法
2. 说明为什么这个功能对用户有价值
3. 等待维护者的反馈

### 提交代码
1. Fork本仓库
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建Pull Request

## 📝 代码规范

- 使用有意义的变量名和函数名
- 添加必要的注释
- 保持代码简洁易读
- 遵循现有的代码风格

## 🧪 测试

在提交PR前，请确保：
- [ ] 扩展在Chrome中正常加载
- [ ] 所有现有功能正常工作
- [ ] 新功能经过测试
- [ ] 没有控制台错误

## 📄 许可

提交代码即表示你同意以MIT许可证发布你的贡献。

## ❓ 问题？

欢迎在Issues中提问！
