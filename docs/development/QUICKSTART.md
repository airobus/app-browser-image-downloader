# 批量图片下载器 - 快速开始

这是一个5分钟快速上手指南。

## 🎯 这个扩展能做什么？

访问任何网页 → 点击扩展图标 → 选择图片 → 批量下载！

就这么简单！✨

## 📦 安装

### 方法1：从Chrome Web Store安装（推荐）
即将上线...

### 方法2：开发者模式安装
1. 下载本项目
   ```bash
   git clone https://github.com/airobus/app-browser-image-downloader.git
   ```

2. 打开Chrome浏览器，访问 `chrome://extensions/`

3. 开启右上角的"开发者模式"

4. 点击"加载已解压的扩展程序"

5. 选择项目文件夹

完成！🎉

## 🚀 使用

### 基础用法
1. **打开扩展**
   - 访问任意网页（如淘宝、京东等）
   - 点击工具栏中的扩展图标

2. **选择图片**
   - 点击图片卡片即可选择/取消
   - 被选中的图片会高亮显示

3. **下载**
   - 点击"下载选中"按钮
   - 图片会自动保存到浏览器默认下载目录

### 高级功能

#### 快速全选
点击"全选"按钮选择页面所有图片

#### 智能筛选
点击"仅大图"自动选择尺寸≥500x500的高质量图片

#### 清除选择
点击"清除选择"取消所有已选图片

## ❓ 常见问题

**Q: 下载的图片保存在哪里？**  
A: 浏览器默认下载文件夹（通常是"下载"或"Downloads"）

**Q: 支持哪些图片格式？**  
A: 支持所有网页图片格式（JPG、PNG、GIF、WebP等）

**Q: 会收集我的数据吗？**  
A: 完全不会！所有操作都在本地完成，不上传任何数据

**Q: 可以修改下载文件名吗？**  
A: 当前版本使用自动命名（image_序号_时间戳.扩展名），未来版本会支持自定义
