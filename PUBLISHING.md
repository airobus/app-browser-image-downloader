## 🚀 发布到 Chrome Web Store

### 1. 准备开发者账号
- 访问 [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole/)
- 注册账号（需支付一次性费用 $5 USD）

### 2. 创建扩展包
```bash
# 在项目根目录执行
# 确保只包含必要文件，排除开发文件
zip -r extension.zip . -x "*.git*" "*.DS_Store" "screenshots/*" "PUBLISHING.md"
```

或者手动选择文件打包：
```
extension.zip
├── manifest.json
├── background.js
├── popup.html
├── popup.js
├── icon16.svg
├── icon48.svg
└── icon128.svg
```

### 3. 上传到商店
1. 登录开发者控制台
2. 点击「新建项目」
3. 上传 `extension.zip`
4. 填写商店信息

### 4. 填写商店信息

#### 基本信息
- **名称**: 批量图片下载器
- **简短描述**: 轻松选择并批量下载网页中的图片，支持跨域下载
- **详细描述**:
```
批量图片下载器是一个简洁高效的浏览器扩展，帮助你快速选择并批量下载网页中的图片。

【主要功能】
✨ 智能扫描 - 自动识别当前网页的所有图片
✅ 批量选择 - 支持单选、全选、反选操作
📏 尺寸过滤 - 一键筛选大图（≥500x500像素）
💾 批量下载 - 支持跨域图片下载
🎨 美观界面 - 现代化设计，操作直观流畅

【使用方法】
1. 访问任意包含图片的网页
2. 点击浏览器工具栏中的扩展图标
3. 选择需要下载的图片
4. 点击「下载选中」按钮

【隐私保护】
✅ 不收集任何用户数据
✅ 不上传任何信息到服务器
✅ 所有操作都在本地完成

开源项目地址：https://github.com/airobus/app-browser-image-downloader
```

#### 分类
- **主要类别**: 生产力工具 (Productivity)
- **次要类别**: 照片 (Photos)

#### 语言
- 中文（简体）

#### 隐私政策
- 粘贴 PRIVACY.md 的内容，或者提供隐私政策网页链接

#### 上传素材
- 上传 icon128.svg (或转换为PNG)
- 上传截图（从 screenshots/ 目录）
- 上传推广图片（可选）

### 5. 提交审核
- 检查所有信息无误
- 点击「提交审核」
- 等待审核（通常需要几天到一周）

## 🔄 版本更新流程

当需要更新扩展时：

1. **修改代码**
2. **更新版本号** - 在 `manifest.json` 中修改 `version`
3. **更新 README** - 记录更新内容
4. **重新打包**
5. **上传新版本到商店**
6. **发布 GitHub Release**

---

## 📝 版本规范

遵循语义化版本控制 (Semantic Versioning):
- `1.0.0` - 主版本.次版本.修订版本
- 主版本：重大功能变更或不兼容的修改
- 次版本：新增功能，向后兼容
- 修订版本：问题修复，向后兼容
