## [1.1.0] - 2026-01-29

### ✨ 新增功能

#### 📸 图片预览功能
- **双击预览**：双击任意图片卡片即可全屏预览原图
- **原图查看**：在预览模态框中查看完整尺寸的原图，准确判断图片清晰度
- **详细信息**：显示图片的尺寸、文件大小和完整 URL
- **文件大小显示**：每张缩略图下方显示文件大小（自动转换为 B/KB/MB）

#### 🎮 丰富的交互方式
- **键盘导航**：
  - `←` `→` 方向键切换上一张/下一张图片
  - `ESC` 键快速关闭预览
- **鼠标操作**：
  - 点击左右两侧的圆形导航按钮 `‹` `›` 切换图片
  - 点击右上角 `×` 按钮关闭预览
  - 点击深色背景区域关闭预览

#### 🎨 视觉优化
- 现代化深色模态框设计（95% 黑色透明度）
- 平滑的淡入动画效果
- 优雅的悬停和交互反馈
- 智能导航按钮（首/尾图片时自动隐藏）

#### ⚡ 性能优化
- 异步获取图片文件大小，不阻塞界面渲染
- 预览时才加载原图，节省资源
- 智能显示"获取中..."加载状态

### 🔧 技术改进
- 新增 `fetchImageSize()` 函数，使用 HEAD 请求获取文件大小
- 新增 `formatFileSize()` 函数，智能格式化文件大小显示
- 实现完整的预览模态框系统（HTML + CSS + JavaScript）
- 添加键盘事件监听和导航控制

### 📝 文件变更
- `popup.html`：新增预览模态框 HTML 结构和完整 CSS 样式（约 120 行样式代码）
- `popup.js`：新增预览功能逻辑和文件大小处理（约 80 行新代码）

## 版本规范

本项目遵循 [语义化版本控制](https://semver.org/lang/zh-CN/)：

- **主版本号 (MAJOR)**: 不兼容的 API 修改
- **次版本号 (MINOR)**: 向下兼容的功能性新增
- **修订号 (PATCH)**: 向下兼容的问题修正
